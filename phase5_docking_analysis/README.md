# Phase 5 — Docking Analysis

## Objective

Dock HRO761 and VVD-214 into WT and mutant WRN pocket models to assess binding-mode preservation and resistance liability.

## Status

**Not started**

## Expected Approach

- Use Phase 2 cocrystal structures as reference binding modes
- Dock inhibitors into Phase 3 mutant models
- Compare pose RMSD, key interaction preservation, clash analysis
- For covalent inhibitors: assess Cys727-mutant loss of covalent engagement

## Expected Outputs

- `docking_results.csv` — Per-inhibitor × per-mutation docking scores and pose metrics
- `pose_comparison_figures/`
- `phase5_summary.md`

## Boundary Conditions

- Docking scores are relative comparisons, not binding affinity predictions
- No claim of clinical efficacy
