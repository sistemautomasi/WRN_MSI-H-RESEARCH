# Phase 5 — Complex PDB files (receptor + ligand merged)

Standalone visualisation-ready **PDB complex files**: each file contains the receptor (chain A) merged with the best Vina pose of the ligand (chain L, residue name `LIG`).

## Why this folder exists

AutoDock Vina writes **ligand-only** pose PDBQTs (`poses/*.pdbqt`) — the receptor is not embedded. To open a docking result in PyMOL, ChimeraX, or Coot you would normally have to load two files and superpose them manually. This folder gives you a single self-contained `.pdb` per docking so any structure viewer opens it correctly out of the box.

## File contents

**Complex PDBs** (68 files, format `<docking_id>_complex.pdb`):
- REMARK header with `docking_id`, `best_score`, `template`, `mutation`, `ligand`, `mode`
- Receptor atoms as `ATOM` records in chain **A** (canonical WRN numbering not applied here — receptor uses template-local numbering, same as the source PDBQT)
- Ligand atoms as `HETATM` records in chain **L**, residue name `LIG`, residue number 1
- Ligand atoms have unique names (`C1, C2, ..., N1, N2, ..., O1, O2, ...`) so PDB parsers count them correctly

**Blocked notes** (6 files, format `<docking_id>_BLOCKED.txt`):
- Text notes for the 6 covalent-blocked dockings (C727A/S vs VVD-214 in covalent-adduct mode)
- No PDB file — these are auto-tagged as chemistry-rule resistance (no thiol nucleophile), no Vina run was performed

**Index** (`README_complexes_index.csv`): row-per-docking with output filename and atom counts.

## Total: 68 PDB + 6 note = 74 files

Matches `phase5_master.csv` (74 rows).

## How to open in PyMOL

```pymol
load complexes/C727R_9S18_HRO761_complex.pdb, top_signal
show cartoon, top_signal and chain A
show sticks, top_signal and chain L
select pocket, (chain A) within 5 of (chain L)
show sticks, pocket
color cyan, pocket
color magenta, chain L
zoom chain L, 5
```

## How to open in ChimeraX

```chimerax
open complexes/C727R_9S18_HRO761_complex.pdb
hide #1 target abcs
show #1/A cartoons
show #1/L atoms
color #1/L byhet
zoom #1/L
```
