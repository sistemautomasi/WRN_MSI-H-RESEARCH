# Phase 4 — Pocket Geometry Analysis

## Objective

Quantify how resistance mutations reshape the inhibitor-binding pocket across WT and mutant WRN structures.

## Status

**Not started**

## Expected Approach

- Define pocket using covalent (Cys727) and non-covalent (HRO761) reference ligand coordinates
- Compute pocket volume, surface area, shape descriptors
- Map electrostatic changes at mutation sites
- Compare pocket geometry across conformational states (apo / inhibitor-bound / nucleotide-bound)

## Expected Outputs

- `pocket_geometry_table.csv` — Per-structure × per-mutation pocket metrics
- `pocket_volume_comparison.png`
- `electrostatic_difference_maps/`
- `phase4_summary.md`
